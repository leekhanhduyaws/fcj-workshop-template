### Week 9 Objectives:

* Develop **FlashLearn**, a web-based English learning platform featuring flashcards, quizzes, battle mode, and a collaborative learning community.
* Deploy a modern AWS architecture in the Singapore Region using a Virtual Private Cloud (VPC) spanning two Availability Zones to ensure high availability.
* Optimize application performance by integrating a serverless architecture with managed AWS services such as Amazon CloudFront, AWS Lambda, and Amazon EventBridge.

### Week 9 Achievements:

* Completed the network architecture, including a Virtual Private Cloud (VPC), public and private subnets, an Application Load Balancer (ALB), and synchronous replication for Amazon RDS PostgreSQL across multiple Availability Zones.

* Successfully utilized Amazon S3 and Amazon CloudFront to deliver static web content efficiently.

* Integrated Amazon Polly to provide text-to-speech (TTS) functionality through internal API calls.

* Configured Amazon EventBridge together with AWS Lambda to automate scheduled tasks, including learning reminder notifications.

* Conducted team meetings to finalize the PostgreSQL database schema design and enhance the user interface in Figma.

* Strengthened expertise in AWS security and identity management services, including AWS Identity and Access Management (IAM), AWS WAF, and NAT Gateway, to ensure a secure and reliable system architecture.
