### Kết quả đạt được tuần 1:

* Hiểu được các khái niệm nền tảng về điện toán đám mây và hệ sinh thái dịch vụ AWS.

* Nắm được chức năng của các nhóm dịch vụ chính trên AWS:
  * Compute
  * Storage
  * Networking
  * Database
  * Security

* Tạo thành công tài khoản AWS Free Tier và làm quen với AWS Management Console.

* Cài đặt và cấu hình thành công AWS CLI, bao gồm:
  * Access Key
  * Secret Access Key
  * Default Region
  * Output Format

* Thực hiện được các thao tác cơ bản bằng AWS CLI như:
  * Kiểm tra cấu hình CLI
  * Liệt kê các AWS Regions
  * Quản lý Key Pair
  * Kiểm tra thông tin EC2
  * Truy vấn thông tin tài khoản AWS

* Khởi tạo thành công Amazon EC2 Instance, kết nối SSH và quản lý EBS Volume.

* Hình thành nền tảng kiến thức và kỹ năng sử dụng AWS Console kết hợp với AWS CLI để chuẩn bị cho các nội dung thực hành ở các tuần tiếp theo.
