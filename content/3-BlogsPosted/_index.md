This section provides a list and an introduction to the blog posts that have been published on the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

###  [Blog 1 - ENHANCING SECURITY WITH SESSION POLICIES IN AMAZON EKS POD IDENTITY](3.1-Blog1/)
This blog explores Session Policies in Amazon EKS Pod Identity and explains how this feature can be used to manage IAM permissions for individual Kubernetes Pods. By using Session Policies, organizations can reduce the number of IAM Roles that need to be created while supporting the implementation of the Principle of Least Privilege (PoLP), thereby enhancing security and simplifying permission management.

###  [Blog 2 - BUILDING EVENT-DRIVEN ARCHITECTURES ON AWS WITH AMAZON EVENTBRIDGE, AMAZON SNS, AND AMAZON SQS](3.2-Blog2/)
This blog explores the Event-Driven Architecture (EDA) pattern and explains how services communicate through events instead of direct service-to-service interactions. It also introduces the roles of Amazon EventBridge, AWS Lambda, Amazon SQS, and Amazon SNS in building highly scalable systems, reducing dependencies between components, and optimizing operational costs.
